📖 About The Project

Traditional portfolios often present information in a static format. This project takes a different approach by combining modern web development with 3D graphics to create a more engaging and memorable experience.

Built using React and Three.js, the portfolio features interactive 3D elements, smooth page transitions, animated sections, and a responsive design that works across devices. The goal was not only to showcase my work but also to demonstrate my ability to build visually appealing and technically sophisticated web applications.

⸻

✨ Key Features

* 🎨 Interactive 3D environment powered by Three.js
* ⚡ Smooth animations and transitions using Framer Motion
* 📱 Fully responsive design for desktop and mobile devices
* 🛠 Dynamic skills and technology showcase
* 📂 Dedicated projects section with detailed project cards
* 📄 Professional experience timeline
* 📬 Contact form integration for direct communication
* 🚀 Fast development and optimized performance with Vite
